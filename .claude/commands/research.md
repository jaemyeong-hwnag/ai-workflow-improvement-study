"$ARGUMENTS" 주제에 대해 최신 논문, 공식 문서, 벤치마크를 조사하고 관련 문서에 반영한다.

## 조사 방법

1. 기존 `docs/` 문서들을 읽어 현재 레퍼런스 현황을 파악한다.
2. 다음 소스에서 관련 정보를 조사한다:
   - arXiv (최신 논문)
   - Anthropic 공식 문서 및 블로그
   - GitHub 공식 레포 (Claude Code, anthropics)
   - SWE-bench, HumanEval 등 AI 코딩 벤치마크
   - 관련 학술 논문 (ACL, NeurIPS, ICML 등)
3. 조사한 레퍼런스를 평가한다:
   - 발행 연도 및 신뢰도
   - 이 레포의 주제와 관련성
   - 기존 문서에서 이미 언급된 것인지 확인
4. 새로운 인사이트나 수치가 있다면 관련 문서를 업데이트한다.
5. `README.md`의 주요 참고 자료 테이블을 업데이트한다.

## 출력 형식

- 새로 발견한 레퍼런스 목록 (제목, 출처, 핵심 내용 한 줄)
- 어느 문서를 업데이트했는지
- 기존 내용과 충돌하거나 업데이트가 필요한 부분이 있다면 명시

## 주의사항

- 확인되지 않은 수치나 주장을 추가하지 않는다
- 논문 arXiv ID 또는 DOI를 함께 기록한다
- 웹 검색 결과 URL은 실제 접근 가능한 것만 포함한다
